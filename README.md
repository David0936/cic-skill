## 这是什么

这是一个 [Claude Code Skill](https://docs.anthropic.com/claude/docs/claude-code-skills)，
通过 [女娲 · Skill造人术](https://github.com/alchaincyf/nuwa-skill) 蒸馏生成。

加载后，Claude 可以用关娜西的视角分析问题——不是模仿她说话，
而是**用她的心智模型 + 决策启发式 + 表达 DNA** 替你拆解一个具体场景。

## 安装与使用

### 安装

```bash
# 方法 1：克隆到 Claude Code 全局 skills 目录
git clone https://github.com/David0936/CIC.skill.git \
  ~/.claude/skills/cc-guannaxi-perspective
```

或者直接下载 zip 解压到 `~/.claude/skills/cc-guannaxi-perspective/`。

### 激活

在 Claude Code 对话框输入任意一项即可触发：

- 最简唤醒：`【CC】` 或 `【关娜西】`
- 自然语言：「关娜西会怎么看……」「问问 CC……」「扮演小C老师……」
- 关键词：「Dr.CiC」「微美人 Barbie」「拼命三姐」「自己成为豪门」「上行电梯论」

### 示例

```
【CC】我做美甲 5 年，国内卷不动了，要不要学你搬去海外重做账号？
```

Claude 会以关娜西的语气和方法论给出 A/B 表格化的分析与具体动作建议。
